# Project Berlin Clock

The app shows how to implement simple Berlin clock 

## Getting Started

Install sdk man
* [SDK Man](https://sdkman.io/install)

Install grails

 * skd i grails 3.1.4
 
Clone the app   
 
 * git clone git@github.com:osvadimos/berlin.git
 
Go to app 

 * cd berlin
 
Run tests

 * grails test-app
